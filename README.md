# Premium Portfolio

A Hollywood-grade, cinematic personal portfolio built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, and tsParticles.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
